# Приклад: базовий codegen

## 1) Запускаємо codegen

```bash
npx playwright codegen https://demo.playwright.dev/todomvc
```

## 2) Записуємо простий сценарій

Наприклад:
- додати 2 todo
- відмітити 1 як виконаний
- перевірити лічильник

## 3) Після запису — робимо “прибиральну” роботу

### ✅ Робимо
- замінюємо `locator('css...')` на `getByRole/getByLabel/getByTestId`
- прибираємо зайві `click()` по декоративним елементам
- додаємо читабельні імена змінних

### ❌ Не робимо
- не залишаємо `waitForTimeout`
- не тримаємо дуже довгі ланцюжки селекторів
