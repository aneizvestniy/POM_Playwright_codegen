# 01 — Codegen (Playwright)

## Мета модуля

Навчитись швидко стартувати з Playwright через **codegen**:
- згенерувати базовий тест
- отримати стабільні локатори
- згенерувати прості assertions
- перевести “записаний” тест у читабельний та підтримуваний код

## Що таке codegen

Playwright Codegen — це режим запису дій у браузері з паралельною генерацією коду тесту. Він:
- підбирає локатори з пріоритетом на **role-based** та семантичні (коли можливо)
- намагається зробити локатор **унікальним** (якщо елементів кілька)
- дозволяє одразу згенерувати **assertions**

## Команди

### Запис тесту

```bash
npx playwright codegen https://demo.playwright.dev/todomvc
```

### Корисні поради

- Записуйте **короткі сценарії** (5–15 кроків), не намагайтесь записати “все” одним тестом.
- Після запису:
  - винесіть повторювані дії в helper / Page Object
  - замініть "жорсткі" локатори на `getByRole/getByLabel/getByTestId`
  - приберіть дублювання `page.goto()` та неочевидні waits

## Типові проблеми та як виправити

- **Надто довгі локатори** → спростити (testid/role), або додати у продукт атрибути `data-testid`.
- **Флейки через timeouts** → замінити `waitForTimeout` на `expect(locator).toBeVisible()`.
- **Селектори на CSS/XPath** → лишати як крайній варіант (коли нема семантики).
