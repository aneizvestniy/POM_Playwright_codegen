# Вправи — 03 Організація проєкту

## Вправа 1 — Розкласти код по папках

Дано: `tests/` містить Page Objects, компоненти і утиліти впереміш.

Завдання:
- розклади по папках `src/pages`, `src/components`, `src/utils`, `tests`
- онови імпорти

## Вправа 2 — Домовленості

Сформулюй правила для команди (коротко):
- де зберігаються локатори
- де зберігаються asserts
- як називаються файли

> Це буде міні-"contributing" правило для автотестів.
