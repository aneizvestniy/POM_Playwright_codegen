# Вправи — 02 POM

## Вправа 1 — Винести локатори в Page Object

Є тест, який напряму використовує `page.locator()` у багатьох місцях.

Завдання:
1) Створи `BasePage` + `LoginPage`
2) Перенеси локатори та дії в `LoginPage`
3) У тесті залиш тільки бізнес-кроки

Критерії:
- у тесті немає CSS/XPath
- мінімум дублювання

## Вправа 2 — Компонент

1) Створи `LoginForm` як компонент
2) Використай його в `LoginPage`
3) Перевір, що зміна локатора у компоненті потребує правки лише в одному місці

## Вправа 3 — Chaining

1) Реалізуй `GuestsListPage.clickAddGuestButton(): AddGuestPage`
2) У тесті зроби ланцюжок: `open()` → `clickAddGuestButton()` → `fillForm()` → `submit()`
3) Додай 1 перевірку переходу (URL або видимість ключового елемента)
