# Презентації

Ця папка містить презентації у форматі Markdown (Marp-сумісні), які відповідають модулям/темам цього репозиторію.

## Як користуватись

- Файли назвемо за схемою: `NN-topic.md` (наприклад: `01-pom-intro.md`).
- Експорт у PDF/PPTX можна робити через наш генератор (Marp) або будь-який інший інструмент.

> Якщо ти скажеш, які саме модулі/нумерація потрібні — я додам перші презентації та структуру під курс.
